Welcome to Draught.
New
·A near-empty canvas. One centered column, a quiet header, and the word count as the only visible control. The writing is the interface.
·It quietly remembers. Make a file a project and Draught keeps versions as you write. Click the word count to see how the work grew, read any past version, or bring one back.
·The folder is your tabs. Every file in the folder sits on the top line, so you can see what you’re working on without going looking. A green dot means the folder is tracked.
·A screenplay lens. Fountain formatting that falls out of typing — the file stays plain text, and Tab is only the escape hatch.
·Scraps and Notes. Cut text lands in a per-manuscript drawer to reuse later; a Notes panel keeps the thoughts beside the draft.
·Exports beautifully. A clean reading page, a submittable manuscript, a book, or a proper screenplay, each as a PDF.
